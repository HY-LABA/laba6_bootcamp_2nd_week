# GitHub 학생 개발자 팩(Student Developer Pack) 가입 가이드

> ⚠️ **이 가이드는 틀릴 수 있습니다.** GitHub의 가입 화면과 절차는 수시로 바뀝니다.
> 가이드대로 하다가 막히거나 화면이 다르게 보이면, **AI에게 질문하거나 인터넷에서 최신 자료를 검색**해 보세요.
> 가장 정확한 정보는 항상 공식 페이지 [education.github.com](https://education.github.com) 에 있습니다.

## 이게 뭔가요?

GitHub가 재학생에게 제공하는 무료 혜택 모음입니다. 주요 항목:

- **GitHub Pro 무료** (비공개 저장소 무제한 등)
- **GitHub Copilot 무료** (AI 코드 자동완성)
- **JetBrains 제품 무료** (IntelliJ, PyCharm 등)
- 기타 여러 개발 도구 무료/할인

전체 혜택 목록은 공식 페이지에서 확인:
👉 [https://education.github.com/pack](https://education.github.com/pack)

## 가입 절차

### 1) GitHub 계정 준비
계정이 없다면 [github.com](https://github.com) 에서 회원가입.

### 2) 학교 이메일을 GitHub에 등록 (학교 이메일로 신청할 경우)
1. GitHub 로그인 → 우측 상단 프로필 아이콘 → **Settings**
2. 왼쪽 메뉴 **Emails**
3. **Add email address** 에 학교 이메일 입력 → **Add**
4. 학교 이메일 받은편지함에서 GitHub 인증 메일의 링크 클릭

### 3) 학생 팩 신청서 제출
👉 [https://education.github.com/discount_requests/application](https://education.github.com/discount_requests/application)

- 학교 이름, 사용 목적, 이메일 등 양식에 따라 입력
- 학교 이메일이 없으면 **학생증/재학증명서를 업로드**하는 옵션이 제시됨

### 4) 결과 대기
신청한 GitHub 계정 이메일로 결과 통보가 옵니다.

## 승인된 후 해야 할 일

### 1) Pro 가입 여부 확인
본인 GitHub 프로필 페이지에 접속하면, **왼쪽 사이드바의 `Highlights` 섹션에 `PRO` 배지**가 추가되어 있는지 확인.

### 2) GitHub Copilot 활성화
👉 [https://github.com/settings/copilot](https://github.com/settings/copilot)

위 페이지에서 활성화 버튼을 누르면 됩니다. 이후 VS Code에 **GitHub Copilot** 확장 설치 후 로그인.

### 3) 그 외 혜택 둘러보기
👉 [https://education.github.com/pack](https://education.github.com/pack)

## 막혔거나 거절(Rejected)되었을 때

- **재신청 가능**합니다. 같은 계정으로 다시 제출하면 됩니다.
- 학생증/재학증명서 사진이 흐릿하면 다시 선명하게 찍어서 시도.
- 학교 이메일이 가장 통과율이 높은 방법이므로, 가능하면 학교 이메일을 등록한 뒤 재신청.
- 상세한 거절 사유와 재신청 안내는 통보 메일 본문에 적혀 있습니다.
